# **A1. 並行鍵場の関手的構成（Functorial Construction of K）**

本文では並行鍵 \(K\) を接束の自己同型として定義したが、  
ここでは **関手的視点** から \(K\) を構成する。

## **A1.1 カテゴリ的背景**
- 対象：滑らかな多様体 \(M\)  
- 射：微分同相写像 \(f : M \to M\)  
- 接束関手：  
  \[
  T : \mathbf{Diff} \to \mathbf{VectBund}
  \]

このとき、並行鍵 \(K\) は  
\[
K : TM \to TM
\]
という **自然変換** として理解できる。

## **A1.2 自然性条件**
任意の微分同相 \(f\) に対し、
\[
T(f) \circ K = K \circ T(f)
\]
が成立する場合、\(K\) は **自然変換としての並行鍵** と呼ばれる。

本文ではこの自然性を明示しなかったため、Appendix で補足する。

# **A2. PKGF における接続の分解と非可換性の測度**

本文では接続 \(\nabla\) を前提として扱ったが、  
ここでは PKGF に特有の **非可換性の測度** を導入する。

## **A2.1 接続の分解**
任意の接続は
\[
\nabla = \nabla^{\text{sym}} + \nabla^{\text{tors}}
\]
に分解される。

- 対称部分：幾何学的整合性  
- ねじれ部分：非可換性の源泉

## **A2.2 非可換性テンソル**
\[
\Theta(K,\Omega) := \nabla K - [\Omega, K]
\]

本文では整合方程式の形で現れたが、  
ここでは **非可換性の測度としてのテンソル** として定義する。

- \(\Theta = 0\)：整合状態  
- \(\Theta \neq 0\)：構築・解体・代謝の駆動源

# **A4. PKGF の時間方向接続の幾何学的補題**

本文では \(\nabla_t\) を導入したが、  
ここではその **幾何学的性質** を補題として整理する。

## **補題 A4.1（時間方向のホロノミー）**
\[
\nabla_t K = 0 \quad \Longleftrightarrow \quad K(t) = U(t) K(0) U(t)^{-1}
\]
ここで \(U(t)\) は時間方向の並行移動。

## **補題 A4.2（時間方向曲率）**
\[
F_t = \partial_t \omega_t + \omega_t^2
\]

時間方向の曲率がゼロでない場合、  
PKGF の時間発展は非可換となり、ゲージ破れの前兆となる。

# **A5. PKGF のランク構造に関する技術的補題**

本文ではランク跳躍（U6）を扱ったが、  
ここではランクの数学的性質を補足する。

## **補題 A5.1（ランクの半連続性）**
行列場 \(K(x)\) のランクは  
**上半連続** である。

つまり、
\[
\limsup_{t\to t_0} \mathrm{rank}(K(t)) \le \mathrm{rank}(K(t_0))
\]

これは逆PKGFのランク単調性（D3）と整合する。

## **補題 A5.2（ランク跳躍の必要条件）**
ランクが増加するためには、
\[
\det(K_{\text{core}}) = 0
\]
となる特異点を通過する必要がある。

本文では述べなかったが、  
これは **次元跳躍の数学的必然性** を与える。

本文ではゲージ破れを定義したが、

## **A6.1 安定化群の定義**
\[
\mathrm{Stab}(K) = \{ H \in \mathcal{G} : HKH^{-1} = K \}

\[
\mathcal{G}/\mathrm{Stab}(K)

が縮退し、  
**新しい概念座標系が確立される**。
