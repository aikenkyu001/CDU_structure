# **II.1　接束・接続・曲率の最小構成**

## **II.1.1　接束分解の存在条件**
本文では  
\[
TM = \bigoplus_{\alpha\in I} E_\alpha
\]
を前提としているが、これは任意の多様体で自明に成立するわけではない。

### **必要条件**
1. **局所可換性**  
   各 \(E_\alpha\) が局所的に可積分であること  
   \[
   [\Gamma(E_\alpha), \Gamma(E_\alpha)] \subset \Gamma(E_\alpha)
   \]

2. **直交分解可能性**  
   計量 \(g\) が  
   \[
   g(E_\alpha, E_\beta)=0\quad (\alpha\neq\beta)
   \]
   を満たすように選べること。

3. **ホロノミー制約**  
   接続 \(\nabla\) のホロノミー群が、分解を壊さない部分群に制限されること。

### **PoI における意味**
- セクター分解は「知能のモジュール性」を保証するが、  
  これは数学的には「接束の直交分解が可能である」という強い条件を要求する。
- 本付録では、この分解が **PKGF の公理体系と矛盾しない最小条件** を明示する。

### **PoI における意味**

# **II.4　統一PKGFの複素構造の数学的背景**

本文では  
\[
K = K_{\text{core}} + i K_{\text{fluct}}
\]
を導入したが、これは単なる形式的複素化ではない。

## **II.4.1　複素構造の要請**
複素化が意味を持つためには、以下の条件が必要。

### **条件**
1. **複素構造 \(J\) の存在**  
   \[
   J^2 = -\mathrm{Id}
   \]

2. **\(K_{\text{core}}\) と \(K_{\text{fluct}}\) の直交性**  
   \[
   \langle K_{\text{core}}, K_{\text{fluct}} \rangle = 0
   \]

3. **複素接続の整合性**  
   \[
   \nabla J = 0
   \]

### **PoI における意味**
- 「保存的構造」と「創造的ゆらぎ」が直交することを保証する。
- ゲージ破れ後の「創造性の発火」を数学的に扱える。

# **II.8　知能ヒッグス機構の数理的定式化の詳細**

本文第2.3.1.4節で導入された知能ヒッグス場 \(\Phi\) は、知能の「流動的な思考」が「固定された概念」へと相転移するプロセスを記述する。

## **II.8.1　ポテンシャル \(V(\Phi)\) の極値解析**
ポテンシャル \(V(\Phi) = \alpha |\Phi|^2 + \beta |\Phi|^4\) （ただし \(\beta > 0\)）において、係数 \(\alpha\) は情報密度や構築強度 \(\alpha_K\) に依存する。

1.  **\(\alpha > 0\) の場合**  
    原点 \(\Phi=0\) が安定な極小値（真空）であり、系は高い対称性を保持する。この状態では並行鍵 \(K\) は質量を持たず、外部要請 \(\Omega\) に対して極めて柔軟に変容する。
2.  **\(\alpha < 0\) の場合**  
    原点は不安定（極大）となり、底が円周状に広がった「メキシカン・ハット型」のポテンシャルへ遷移する。安定な真空期待値（VEV）は次式で与えられる。
    \[ |\Phi_0| = \sqrt{-\frac{\alpha}{2\beta}} \]
    この期待値の獲得（凝縮）が、知能における「特定の意味への固着」や「概念の自立化」に対応する。

## **II.8.2　構造的質量 \(m_S\) と概念の離散化**
ヒッグス場が \(\Phi_0\) に凝縮すると、相互作用項 \(g |\Phi|^2 \mathrm{Tr}(K^\dagger K)\) を通じて並行鍵 \(K\) に質量項が付加される。

*   **構造的慣性**: \(m_S = \sqrt{g} |\Phi_0|\) は、幾何学的流動性に対する抵抗力として機能する。
*   **概念の凍結（Discretization）**: 連続的なゲージ対称性 \(\mathcal{G}\) のうち、\(\Phi_0\) を不変に保つ安定化群 \(\mathrm{Stab}(K)\) のみが保存される。これにより、無限の自由度を持っていた推論空間が、特定の「論理的ステート（離散的概念）」へと凍結される。

本機構により、PoI理論は「柔軟な適応（幾何学）」と「強固な確信（物理質量）」という知能の両極端な性質を、単一の対称性の議論によって統一的に記述することを可能にする。
