## 1. シミュレーション概要
光学ボケ（PSF）を伴う行列ダイナミクスにおいて、再帰的演算が構造の次元（有効ランク）に与える影響を検証した。

```mermaid
graph TD
    K_t[現在の並行鍵 K_t] --> A{C: 光学ボケ A}
    A --> B[拡散画像]
    B --> C{D: 非線形・散逸}
    C --> K_next[次の並行鍵 K_{t+1}]
    K_next --> K_t
    K_next --> E{U: 安定相への収束}
    E -- rank jump --> F[構造生成成功]

```

## 3. 結論
絶対値に差異（Fortran 版の計算手法による）があるものの、**「ダイナミクスが安定相へ移行し、相対的なランク変化が停止する」という挙動は両者で一致**した。物理実装においては、光学系の収差がこの安定相の形状（attractor）を決定する鍵となる。
